Built to Solve Real Problems

AcadMark - Smart Attendance System

Problem: Manual attendance is slow, error-prone & hard to analyze.

Solution: A role-based attendance platform where teachers start a time-bound session, students mark attendance in a 3-minute window & admins monitor performance using dashboards & defaulter analytics.

  • Teacher-controlled live session with 3 minute attendance window
  • Student attendance marking with secure role access
  • Admin dashboard with defaulter list & trend visibility
  • Attendance stats & analytics for better decisions

Tech: Node.js, Express.js, MySQL

Your Role: Full-stack product developer with backend ownership.
